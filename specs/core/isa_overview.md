Overview

This ISA is a security-first architecture inspired by RISC-V, designed around capability-based access, deterministic execution, and hardware-enforced isolation.

Design Principles
Security by default (not opt-in)
Deterministic and auditable execution
Minimal trusted computing base
Composability over complexity
Key Features
Capability-based memory access
Tagged memory
Hardware isolation domains
Optional Information Flow Control (IFC)
Deterministic execution mode
