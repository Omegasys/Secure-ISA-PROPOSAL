# 64-bit Variant (Baseline)

## Overview

The 64-bit configuration is the **primary target architecture**.

## Characteristics

* Register width: 64 bits
* Address space: 16 exabytes (theoretical)
* Capability size: 128 bits

## Register Model

* 32 general-purpose registers (64-bit)
* 16 capability registers (full precision)

## Memory Model

* 64-bit virtual addressing
* Tagged memory with full metadata

## Performance Profile

* Balanced performance and security
* Suitable for general-purpose systems

## Security Features

* Full capability precision
* Strong isolation guarantees
* Efficient audit support

## Extensions

* Vector/SIMD support
* Cryptographic acceleration
* Virtualization support

## Use Cases

* Operating systems
* Secure servers
* High-assurance computing environments
